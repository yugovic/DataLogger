## プロダクト概要

車両のサーキット走行セットアップ（タイヤ空気圧・サス・アライメント等）を記録・管理し、
検証済みセットアップデータの共有・売買を行うプラットフォーム。
テーマは「**データを使ってモータースポーツをより楽しくする**」
（スポーツとして楽しむ / 高みを目指す / 成長していく）。

事業の堀は機能ではなく「複製不可能な検証済みデータ資産」。
無料の記録機能＝データの仕入れ部門であり、記録体験とデータ品質が生命線。

## 技術的制約

- React 18 + TypeScript + Tailwind + Ant Design + Firebase（Firestore/Auth/Storage）
- 未入力は null 保存（0変換・デモ初期値は禁止 — 偽データ混入は事業上の欠陥）
- スペック値は「申告値」であり、証憑（ロガー計測）と区別して表示する
- 共有スナップショットには車両写真を含めない（個人特定・権利面の配慮）
- バンドルサイズ 2.8MB が既知の課題。重い追加は要検討
