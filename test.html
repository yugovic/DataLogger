CarSetup6 開発サーバー接続テスト

手順：

  1. ターミナルで以下のコマンドを実行： 
    cd /Users/Yugox/Documents/Program/CarSetup6
npm run dev
  2. 開発サーバーが起動したら、表示されるURLにアクセス（通常は http://localhost:5173）
  3. ブラウザの開発者ツール（F12キー）を開いて、コンソールタブでエラーを確認

確認事項：

  • 白い画面が表示される場合は、コンソールにエラーがないか確認
  • 「VELOCITY LOGGER」のヘッダーが表示されれば、SimpleModernCarSetupは正常動作
  • エラーが表示される場合は、そのエラーメッセージをコピー

トラブルシューティング：

  • npm run devでエラーが出る場合： npm install を実行
  • ポートが使用中の場合： 別のポートを使用するか、既存のプロセスを終了
  • 認証エラーの場合： Firebaseの設定を確認
